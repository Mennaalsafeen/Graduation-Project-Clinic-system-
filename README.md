## 📌 Project Overview
A web-based application built with **ASP.NET Core MVC** designed to streamline dental appointment booking. The system aims to eliminate scheduling conflicts and improve the experience for both patients and clinic staff.

### ⚠️ Problem Statement
Manual and phone-based booking lead to:
* Scheduling conflicts and double bookings.
* Inefficient time management for dentists.
* Poor patient experience due to waiting times.

### 🎯 Project Objectives
1. **Online Booking:** 24/7 appointment scheduling for patients.
2. **Efficiency:** Organized dashboard for clinic staff.
3. **Visibility:** Personalized schedules for dentists.
4. **Workflow:** Overall optimization of clinic operations.

## 📋 System Specifications

### 📍 Scope
* **In Scope:** Authentication, Booking, Appointment Management, Admin Dashboard.
* **Out of Scope:** Online Payments, e-Prescriptions, Full EMR.

### ✅ Functional Requirements
* User registration and secure login.
* Self-service booking/canceling for patients.
* Real-time appointment tracking.
* Dentist availability management.
* Administrative control over users and schedules.

### ⚡ Non-Functional Requirements
* **Usability:** Intuitive and user-friendly interface.
* **Performance:** Fast response times for booking actions.
* **Security:** Data privacy and secure user sessions.
* **Responsiveness:** Support for mobile and desktop screens.
