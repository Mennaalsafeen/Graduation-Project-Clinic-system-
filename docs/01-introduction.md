# Introduction

## Problem Statement
Dental clinics often rely on manual or phone-based appointment booking, which may cause scheduling conflicts, missed appointments, and wasted time. A centralized system is needed to organize appointments and improve communication between patients, dentists, and clinic staff.

## Project Objectives
- Allow patients to book dental appointments online.
- Help clinic staff manage and organize schedules efficiently.
- Enable dentists to view and manage their appointments.
- Reduce waiting time and appointment conflicts.
- Improve the overall clinic workflow and user experience.

## Scope
### In Scope
- User registration and login
- Appointment booking
- Appointment management
- Dentist schedule viewing
- Admin management of users and appointments

### Out of Scope
- Online payment processing
- Electronic prescriptions
- Full electronic medical record management
- Integration with external hospital systems

## Assumptions & Constraints
- The project will be developed by a small team within a limited time frame.
- The system will use ASP.NET Core MVC and SQL Server.
- The first release will focus on basic appointment management features only.
- Advanced dental clinic features will not be included in Week 1.
- Users are expected to have internet access to use the system.
