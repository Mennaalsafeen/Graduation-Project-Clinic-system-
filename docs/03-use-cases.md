# Use Cases

## System Roles
- Patient
- Dentist
- Admin / Receptionist

## Patient Use Cases
- Register a new account.
- Log in to the system.
- Book a dental appointment.
- View upcoming and past appointments.
- Cancel or reschedule an appointment.

## Dentist Use Cases
- Log in to the system.
- View assigned appointments.
- Manage daily schedule and availability.
- Review appointment details.

## Admin / Receptionist Use Cases
- Log in to the system.
- Manage users and accounts.
- Manage appointments.
- Assign or update dentist schedules.
- Monitor clinic booking activity.

## Simple Scope Diagram
Guest / Patient / Dentist / Admin → Dental Clinic Appointment Management System
