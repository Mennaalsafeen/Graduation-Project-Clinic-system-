# File 08: 08_STITCH_PROMPT.md

> **ONLY for Google Stitch — generates all 18 UI pages from this design prompt.**

## 1. Project Identity
- **Project Name:** Dental Clinic Appointment Management System
- **Core Goal:** Digitalize clinic workflows with a professional Arabic interface (E-Business Integration).

## 2. Visual Identity & UI Standards
- **Design Style:** Blue/Teal theme (#008080), Clean White backgrounds (#F8F9FA).
- **Components:** Rounded cards (10px radius), Professional shadows, Modern icons.
- **Responsiveness:** Full Mobile/Tablet/Desktop support.
- **Language:** Strict RTL (Arabic) layout as the primary interface.

## 3. Project Scope (Exactly 18 Pages)
Generate high-fidelity UI views for the following 18 pages:

### A. Public Area (5 Pages)
1. **Landing Page:** Hero section with "Dental Clinic Appointment Management System" branding and CTA.
2. **Login:** Secure sign-in with 2FA verification placeholder.
3. **Register:** Patient registration form (National ID, Phone, Name).
4. **Services:** Treatment grid showing services and prices.
5. **Contact Us:** Clinic information and contact form.

### B. Patient Portal (4 Pages)
6. **Patient Dashboard:** Summary of upcoming appointments and health alerts.
7. **Booking Wizard:** High-performance single-page booking system.
8. **My Visits:** History of treatments and digital invoices.
9. **Feedback:** Rating and comment form (Essential for E-Business analytics).

### C. Doctor Portal (3 Pages)
10. **Doctor Dashboard:** Daily stats and monthly performance charts.
11. **Work Schedule:** Today's appointments queue and schedule management.
12. **Clinical Hub:** Patient record with interactive SVG Dental Chart (32 teeth).

### D. Admin & Operations (6 Pages)
13. **Admin Dashboard:** Global clinic analytics (Revenue and Patient volume).
14. **Staff Management:** Control panel for Doctors and Receptionists.
15. **Inventory:** Medical supplies tracking with low-stock alerts.
16. **Financials:** Revenue reports, service analytics, and debt tracking.
17. **Audit Trail:** Security logs for tracking every system action.
18. **Receptionist Flow:** Real-time clinic occupancy and quick check-in tool.

## 4. Instructions for Stitch
- **Action:** Generate all 18 pages using the "Dental Clinic Appointment Management System" branding.
- **Technical:** Use Bootstrap 5 and ensure all components are RTL-ready for Arabic.
