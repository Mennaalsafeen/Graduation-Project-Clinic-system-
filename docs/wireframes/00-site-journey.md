# Site Journey

## Guest User Flow(Non-Registered User Path)
- Home / Landing Page
  - Main page showing clinic services (cleaning, fillings, orthodontics, etc.)
- Login Page
  - Allows registered patients to log in using email/username and password
- Register Page
   - Create a new patient account (name, email, password, phone, etc.)
- Forgot Password Page [optional / not implemented yet]

- Detailed Description:  
  - Home: Displays an overview of dental services such as cleaning, fillings, orthodontics, and general dental care.  
  - Login: Includes fields for email/username and password with a login button.  
  - Register: Fields include full name, email, password, phone number, and password confirmation.  
  - Browse services: Users can view basic info about dental services, doctors, and clinic details without logging in.  
  - View doctors: Displays available dentists with a brief description of experience and specialties.  
  - Undefined pages: Mark any unclear pages with [?] for later clarification.

## Logged-in Patient Flow
- Dashboard
  - Shows summary of patient activity (upcoming appointments, past visits, notifications)
- View Appointments
   - Allows patients to see all booked dental appointments
- Book Appointment
 - Page for booking new appointments (choose service, dentist, date, time)
- Edit Appointment
 - View basic dental history, treatments, or notes from previous visits
- Cancel Appointment
- Logout
- Detailed Description:  
  - Home: May display personalized info such as upcoming appointments or recommended services.  
  - Dashboard: Summary of patient activity, appointments, and notifications.  
  - View appointments: Allows patients to check their booked appointments.  
  - Book appointment: Users select service, dentist, date, and time to book an appointment.  
  - Medical record: View dental history and previous visit notes.  
  - Logout: Button to sign out of the account.  

## Admin / Receptionist Flow
- Admin Dashboard
- Manage Users
- Manage Dentists
- Manage Appointments
- Detailed Description:  
  - Admin can manage users, dentists, and appointments efficiently.  
  - The dashboard shows stats and alerts for upcoming appointments.

## Dentist Flow
- Dentist Dashboard
- View Schedule
- View Appointments
- Update Availability
  - Detailed Description:  
  - Dentist can see their schedule, upcoming appointments, and update availability.  
  - Can add notes for patient visits.
